Skill: Bug Triage & Root Cause Analysis (RCA)

1. Descrição

Transforma logs de erro brutos em planos de ação técnicos.

2. Workflow

Receber Log/Stack Trace do usuário.

Localizar o arquivo e linha provável do erro.

Criar uma task em .agents/docs/tasks/ com o diagnóstico e a proposta de correção.

Acionar a Feature Initialization Skill com o tipo fix.
