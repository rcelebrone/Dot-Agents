Skill: Documentation & README Updater

1. Descrição

Garante que o README.md e os arquivos em /docs reflitam a realidade atual do código.

2. Workflow

Sempre que houver mudanças em env variables, atualizar o README.md.

Se houver novos endpoints ou componentes públicos, gerar documentação de uso.

Sincronizar diagramas (Mermaid.js) se houver mudança de fluxo.
